# nestjs-todo-app
## Description

A [Nest](https://github.com/nestjs/nest) framework sample code for my self study.

## Environment

- Node.js v18.15.0
- pnpm v7.29.3

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
